# khetala-server

## Running the server
Clone the repo:
``` sh
$ git clone git@github.com:Data-Crunch-Team-Green/khetala-server.git
```

Install the libraries
``` sh
$ pip install -r requirements.txt
```

Run the server
``` sh
$ uvicorn main:app --reload
```
The Uvicorn server runs on http://127.0.0.1:8000
